About Me

Welcome to my web developer portfolio! I'm Alen Kiyanovski, a dedicated student at CareerFoundry's web development bootcamp, specializing in the MERN stack.

With hands-on experience, I've built full-stack web applications using the MERN stack. Proficient in front-end technologies like HTML, CSS, and JavaScript, I create dynamic and interactive user interfaces with React.

On the back-end, I've worked extensively with Node.js, Express.js, and MongoDB, developing RESTful APIs and mastering authentication, authorization, and data manipulation.

Collaborating on real-world projects at CareerFoundry, I've honed my problem-solving skills and embraced clean code practices. Passionate about creating seamless user experiences, I constantly explore new libraries and frameworks.

Excited to showcase my projects and contribute to innovative web development, feel free to explore my portfolio and connect with me for collaborations or inquiries.
